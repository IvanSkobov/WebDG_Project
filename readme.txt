# 🌐 ZeroCoder Site — DJ02

Простой сайт на Django с 4 страницами, подключённым Bootstrap и компонентами шаблонов (меню и подвал). Стартовая заготовка для портфолио, лендинга или любого учебного проекта.

## 🔧 Технологии

- Django 4+
- Bootstrap 5
- HTML5
- Python 3.10+
- SQLite (встроено)
